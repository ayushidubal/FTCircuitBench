# Miscellaneous Scripts

This folder contains personal-use scripts and notebooks that are not part of the main FTCircuitBench package API. These files are kept for reference but are not intended for public distribution.

## Contents

### Analysis Scripts
- `show_pbc_layers.py` - Display PBC circuit layers and operators
- `inspect_sk_gates.py` - Inspect Solovay-Kitaev gate decomposition
- `run_sk_fidelity_test.py` - Test Solovay-Kitaev fidelity

### Plotting Scripts
- `plot_gs_prec8_stacked_clifford_t_counts.py` - Plot Clifford+T gate counts
- `plot_gs_prec_fidelity_vs_gatecount.py` - Plot fidelity vs gate count
- `plot_gs_prec_histogram_by_stat.py` - Generate histograms by statistics
- `plot_pbc_reduction_by_circuit.py` - Plot PBC reduction metrics

### Utility Scripts
- `export_stats_to_csv.py` - Export statistics to CSV format
- `generate_pipeline_comparison_table.py` - Generate comparison tables

### Notebooks
- `FTCircuitBench_Decompose_integration.ipynb` - Decomposition integration examples
- `FTCircuitBench_Pipeline_Demo.ipynb` - Pipeline demonstration notebook
- `draw_qasm_comparison.ipynb` - QASM comparison visualization

## Note

These files are preserved for personal reference and may be removed before publication if they're not needed. Review each file and decide whether to keep, update, or delete before finalizing the package.
